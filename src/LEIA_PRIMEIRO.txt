🌐 FAZER DEPLOY NO GITHUB PAGES:

1. Crie repositório no GitHub
2. Edite vite.config.ts (altere o 'base')
3. Execute: npm run deploy
4. Ative GitHub Pages nas configurações
5. Pronto! Seu site estará online

Ver DEPLOY.md para instruções detalhadas.

═══════════════════════════════════════════════════════════════

🐛 PROBLEMAS COMUNS:

❌ "npm não é reconhecido"
   → Instale Node.js em: https://nodejs.org/

❌ Erro ao instalar
   → Tente: npm install --legacy-peer-deps

❌ Página em branco após deploy
   → Verifique o 'base' no vite.config.ts

❌ Porta 5173 ocupada
   → Use: npm run dev -- --port 3000

📊 COMO USAR:

1. Prepare seu arquivo Excel com as abas:
   - Testes
   - Conversões
   - Renovações
   - Clientes Ativos
   - Clientes Expirados
   - Jogos (opcional)
   - Conv x Jogos (opcional)

2. Clique em "Importar Excel" no dashboard

3. Navegue pelas 8 views:
   ✓ Overview (visão geral)
   ✓ Financeiro (MRR, ARR, LTV...)
   ✓ Clientes (busca e lista)
   ✓ Retenção (churn, fidelidade)
   ✓ Conversão (taxa, funil)
   ✓ Jogos (impacto de jogos)
   ✓ Geografia (estados, cidades)
   ✓ Tráfego (origem, ROI)
