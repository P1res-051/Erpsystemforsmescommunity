📦 OPÇÃO 2: COMANDOS MANUAIS

═══════════════════════════════════════════════════════════════

1️⃣ Configurar Git (primeira vez):
----------------------------------

git config --global user.name "Seu Nome Aqui"
git config --global user.email "seu@email.com"


2️⃣ Inicializar e fazer primeiro commit:
-----------------------------------------

git init
git add .
git commit -m "🎉 Initial commit - IPTV Dashboard v2.0"


3️⃣ Conectar ao GitHub:
-----------------------

⚠️ ATENÇÃO: Altere SEU-USUARIO e SEU-REPOSITORIO abaixo!

git remote add origin https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
git branch -M main
git push -u origin main


4️⃣ Instalar dependências:
---------------------------

npm install


5️⃣ Fazer deploy para GitHub Pages:
------------------------------------

npm run deploy

🌐 ATIVAR GITHUB PAGES (no site do GitHub)

═══════════════════════════════════════════════════════════════

1. Vá em: https://github.com/SEU-USUARIO/SEU-REPOSITORIO/settings/pages

2. Em "Source", selecione: gh-pages

3. Clique em "Save"

4. Aguarde 2-5 minutos

5. Acesse: https://SEU-USUARIO.github.io/SEU-REPOSITORIO/


═══════════════════════════════════════════════════════════════

🔐 CRIAR PERSONAL ACCESS TOKEN (se pedir senha)

═══════════════════════════════════════════════════════════════

1. Vá em: https://github.com/settings/tokens

2. Click em "Generate new token" → "Generate new token (classic)"

3. Marque: ☑️ repo (todas as opções)

4. Click em "Generate token"

5. COPIE O TOKEN (não será mostrado novamente!)

6. Use o token como SENHA ao fazer git push

🆘 SOLUÇÃO RÁPIDA DE PROBLEMAS

═══════════════════════════════════════════════════════════════

❌ Erro: "git: command not found"
Solução: Instale o Git em https://git-scm.com/


❌ Erro: "npm: command not found"
Solução: Instale o Node.js em https://nodejs.org/


❌ Erro: "Authentication failed"
Solução: Use Personal Access Token como senha


❌ Erro: "Permission denied"
Solução:
chmod +x setup.sh
./setup.sh


❌ Página em branco após deploy
Solução: Verifique o base no vite.config.ts


❌ Erro ao fazer npm install
Solução:
npm install --legacy-peer-deps
